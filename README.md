# Advent of Code

My attempt at building a web app to show off my advent of code solutions. Done in my spare time so I'll focus on functionality first and might come back and add styling.

## Setup

To run the app in dev use `yarn dev`

Prettier and ESLint can be ran using `yarn lint` or separately using `yarn lint:format` and `yarn lint:fix` respectively.

You can also [visit the live web app](https://advent-of-code-rust.vercel.app/)

## Aim

The aim of the project is to have a visual representation of each puzzle and solution. This is obviously a bigger task than just solving the puzzles so the styling and visual representations will take a backseat until I have time to come back and add them
